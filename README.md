💼 JobBoard — React Job Listing Portal

A modern job listing web app built with React + Vite.
Browse 75 high-demand tech jobs, search by role or skills, filter by type/location, and save jobs for later.

⸻

✨ Features
	•	🔍 Search by role, company, or skills
	•	🎯 Filter by job type & location
	•	🔖 Save / unsave jobs
	•	💾 Persistent storage (localStorage)
	•	📊 Live job stats (Full-time, Internship, Remote, Cities)
	•	📱 Fully responsive UI

⸻

🛠️ Tech Stack
	•	React (Hooks)
	•	Vite
	•	JavaScript (ES6+)
	•	CSS (custom styling)
	•	localStorage API
